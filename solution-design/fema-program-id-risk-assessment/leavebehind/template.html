This test lets the historical-mining crawler read only the FY2022–FY2025 mapping history, propose FY2026 groupings blind, then scores those proposals against the held-out FY2026 mapping it was never shown. Inference and scoring are separate code paths — the crawler cannot see the labels it is graded on.
